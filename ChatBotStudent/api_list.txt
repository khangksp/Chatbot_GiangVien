DANH SÁCH 20 API CHÍNH CỦA ỨNG DỤNG SINH VIÊN
===============================================

1. XÁC THỰC & ĐĂNG NHẬP
------------------------
POST /odp/auth/login
- Đăng nhập bằng tài khoản/mật khẩu
- Body: { username, password }
- Response: { token, user: { mssv, email, name, avatar }, expires_in }

POST /odp/auth/google/verify-token  
- Đăng nhập bằng Google SSO
- Body: { token }
- Response: { token, user: { mssv, email, name, avatar }, expires_in }

POST /odp/auth/request-otp
- Gửi mã OTP qua email
- Body: { email }
- Response: { message: "OTP đã được gửi", status: true }

POST /odp/auth/verify-otp
- Xác thực mã OTP
- Body: { email, otp }
- Response: { token, user: { mssv, email, name }, expires_in }

POST /odp/auth/create-account
- Tạo tài khoản mới
- Body: { email, password, ... }
- Response: { message: "Tạo tài khoản thành công", user: { mssv, email } }

GET /odp/auth/me
- Lấy thông tin user hiện tại
- Response: { mssv, email, name, avatar, role, permissions }

POST /odp/auth/logout
- Đăng xuất
- Body: { token, mssv, device_ip }
- Response: { message: "Đăng xuất thành công", status: true }

2. THÔNG TIN SINH VIÊN
-----------------------
GET /odp/sinh-vien/profile
- Lấy thông tin profile sinh viên
- Response: { mssv, ho_ten, email, sdt, lop, khoa, nganh, avatar, ngay_sinh, gioi_tinh, dia_chi }

GET /odp/profile/me
- Lấy thông tin cá nhân (fallback)
- Response: { mssv, name, email, avatar, role }

3. HỌC PHÍ & THANH TOÁN
-----------------------
GET /odp/hoc-phi/tuition-all-by-student
- Lấy danh sách tất cả học phí của sinh viên
- Response: [{ nkhk, ngay_het_han, tong_tien_phai_thu, tong_tien_da_thu, tong_tien_con_lai, status }]

GET /odp/hoc-phi/tuition-by-student?nkhk={nkhk}
- Lấy chi tiết học phí theo học kỳ
- Params: nkhk (niên khóa học kỳ)
- Response: { mssv, nkhk, danh_sach_mon_hoc: [{ ma_mon_hoc, ten_mon_hoc, hoc_phi_mon_hoc, tien_da_thu, ngay_thu, trang_thai_mon_hoc, tin_chi }], tong_tien_phai_thu, tong_tien_da_thu, tong_tien_con_lai, ngay_het_han, trang_thai_tong }

GET /odp/hoc-phi/gen-qr?nkhk={nkhk}
- Tạo mã QR thanh toán học phí
- Params: nkhk
- Response: { qr_code: "base64_image", amount, content: "nội dung chuyển khoản", bank_info: { bank_name, account_number, account_name } }

POST /odp/hoc-phi/upload-tuition?nkhk={nkhk}
- Upload ảnh minh chứng thanh toán
- Params: nkhk
- Body: FormData với file ảnh
- Response: { status: true, message: "Upload thành công", imageTuitionUrl: "https://..." }

GET /odp/hoc-phi/get-image-tuition?nkhk={nkhk}
- Lấy danh sách ảnh minh chứng đã upload
- Params: nkhk
- Response: { tuitionUrl: "https://...", status: "pending|approved|rejected" }

15. ĐIỂM DANH
-------------
GET /odp/diem-danh/all-period
- Lấy thông tin điểm danh tất cả kỳ
- Response: { tong_buoi_hoc: 45, co_mat: 42, vang_mat: 3, ti_le_co_mat: 93.3, chi_tiet: [{ ngay, mon_hoc, trang_thai }] }

GHI CHÚ:
- Tất cả API đều sử dụng prefix "/odp/" (trừ system biểu mẫu)
- Hầu hết API cần token xác thực trong header
- Tham số nkhk (niên khóa học kỳ) được sử dụng rộng rãi
- API upload file sử dụng FormData
- Có hỗ trợ pagination và filtering

CẤU TRÚC RESPONSE CHUNG:
- Thành công: { status: true, data: {...}, message: "..." }
- Lỗi: { status: false, error: "...", code: 400/401/500 }
- Upload: { status: true, imageTuitionUrl: "https://...", message: "..." }

HEADERS YÊU CẦU:
- Authorization: "Bearer {token}"
- Content-Type: "application/json" (trừ upload file)
- Content-Type: "multipart/form-data" (cho upload file)

THAM SỐ QUAN TRỌNG:
- nkhk: Niên khóa học kỳ (format: YYMMK, VD: 25261 = 2025-2026 học kỳ 1)
- mssv: Mã số sinh viên
- ma_nhom: Mã nhóm môn học
- ma_don: Mã đơn/biểu mẫu
